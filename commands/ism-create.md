Create a well-structured GitHub Issue from your idea.

Use the issuesmith-create skill to:
- Guide you from idea to published Issue, section by section
- Auto-detect issue type (feature, bug, perf, docs, refactor) and adapt guidance
- Generate Background, Goal, Non-goals, Acceptance Criteria, Task Checklist, and Notes/Decisions
- Let you review and modify each section before creation
- Create the Issue via `gh issue create` when you confirm

**No local files are written.** The entire process happens in the conversation.
